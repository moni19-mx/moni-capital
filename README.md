# Moni Capital

Dashboard ejecutivo personal de patrimonio (acciones + cripto).

## Correr localmente
npm install
npm run dev

## Desplegar (gratis)
1. Sube esta carpeta a un repo de GitHub.
2. Entra a vercel.com, "Add New Project", conecta el repo.
3. Vercel detecta Vite automaticamente. Deploy.
4. Tu sitio queda en algo como https://moni-capital.vercel.app

## Actualizar datos
Edita las listas STOCKS y CRYPTO en src/App.jsx, sube el cambio a GitHub,
Vercel re-despliega solo.
